
# Quiz API

## Getting Started

- Make sure that you have at leat 3.9 docker version

- Clone the repository and install the necessary dependencies 

```bash
$ git clone https://github.com/adriancielniak/quiz-api.git

$ npm install
 
$ ./install-dependencies.sh
```

## Running the app

```bash
# run database
$ dokcer-compose up -d

# run api
$ npm run start:dev
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
